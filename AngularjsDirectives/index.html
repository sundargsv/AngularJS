Notes

  • * The {{ quantity * price }} expression, in the example above, is an AngularJS data binding expression.
  • * The ng-repeat directive repeats an HTML element
  • * The ng-repeat directive actually clones HTML elements once for each item in a collection.
  • * The directive name is "w3TestDirective", but the Tag/ Element name is "w3-test-directive"
  • * We can also invoke directive by Attribute, Class, Comment <div w3-test-directive> </div> <div class="w3-test-directive"> </div>
  • * You can restrict your directives to only be invoked by some of the methods.
  • * By default the value is EA, meaning that both Element names and attribute names can invoke the directive.
